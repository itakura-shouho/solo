【明確なセットアップ手順】


１．パッケージをインストールする

npm install

２．バックエンドの起動

　npm start

３．フロントエンドの起動

npm run frontend-start


【アプリの詳細】

ユーザーが思ったことを書き留めるブログである。閲覧画面では過去に投稿した記事が閲覧でき、投稿画面では記事の投稿ができる。なお、MVP（2023/6/9現在）では、ブログのタイトルは１０文字以内、本文を３００文字以内と制限している。


【リソース】

React / express / knex / postgres / render
